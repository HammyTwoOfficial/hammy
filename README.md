Installation Instructions
Move the "hammy" folder to your Grasshopper User Objects folder:

Windows: C:\Users\<YourUsername>\AppData\Roaming\Grasshopper\UserObjects

required packages:
    scikit-learn==1.0.2
    numpy==1.26.4
